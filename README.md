## 🛠️ Tech Stack
- **HTML5:** Semantic structure and SEO optimization.
- **CSS3:** Modern styling with Glassmorphism and animations.
- **JavaScript:** Interactive components and multi-language support.
- **Libraries:** 
  - [Particles.js](https://vincentgarreau.com/particles.js/) for background animations.
  - [Typed.js](https://mattboldt.github.io/typed.js/) for dynamic typing effects.
  - [SweetAlert2](https://sweetalert2.github.io/) for professional notifications.
  - [Vanilla-tilt.js](https://micku7zu.github.io/vanilla-tilt.js/) for 3D hover effects.

## ✨ Key Features
- **Multi-language Support:** Seamless switching between English and French.
- **Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, Desktop).
- **Dark/Light Mode:** User-preferred theme persistence.
- **Contact Form:** Integrated with FormSubmit for real-time inquiries.
- **Interactive Skills Showcase:** Animated progress bars and category filtering.
